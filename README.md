# 小而美的个人博客
采用Java编写，根据李仁密的小而美的个人博客系统开发而成

舍弃JPA，采用MyBatis作为持久层框架

框架以及软件版本：
- IDEA ULTIMATE 2019.3
- SpringBoot 2.2.2
- MyBatis 2.1.1
- Thymeleaf 3.0.11
- MySQL 8.0.19
- Redis 5.0.7
- Maven 3.6.3
- JDK 11.0.5
